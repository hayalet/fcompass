# Farewell Compass

> Detect dependencies of your stylesheets to Compass and check if you can say farewell easily

## Install

```
npm install --global fcompass
```

## CLI
```
$ fcompass --help

  Usage
    $ fcompass <glob>

  Example
    $ f-compass "scss/**/*.scss --ruby

  Options
    --ruby  Show only functions requiring ruby
```
